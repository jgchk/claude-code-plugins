# Optimizing Plugins

A Claude Code plugin for creating, auditing, and optimizing Claude Code plugins and skills.

## Installation

Add this plugin to your Claude Code configuration:

```bash
claude plugins:add /path/to/optimizing-plugins
```

Or install from a git repository:

```bash
claude plugins:add https://github.com/username/optimizing-plugins
```

## Usage

The plugin provides the `/optimizing-plugins` skill which activates automatically when:

- Creating new plugins
- Improving existing plugins
- Reviewing plugin structure
- Troubleshooting plugin issues

### Creating a New Plugin

```
Create a new Claude Code plugin for [your use case]
```

### Auditing an Existing Plugin

```
Audit the plugin at /path/to/plugin
```

## Features

- Plugin scaffolding with correct structure
- Comprehensive audit checklist
- Best practice enforcement
- Template generation for skills, commands, and agents
