---
name: weeknote
description: Draft a weeknote - regular team communication about work in progress
---

Draft a weeknote for the current week.

If arguments are provided, use them as context or focus:
- `$ARGUMENTS`

Examples:
- `/weeknote` - Draft this week's update
- `/weeknote focus on the API migration` - Emphasize specific work
- `/weeknote it was a quieter week` - Set the tone
